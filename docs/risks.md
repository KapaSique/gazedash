В данный момент используется MOCK/IN-MEMORY -> после каждого рестарта данные сбрасываются
Проценты зависят от частоты событий и не отражают реальное время без бинирования данных
Нет авторизации/ролей -> любой локальный клиент может читать данные
Схема событий упрощена (type/value/confidence), нет версии протокола
Duration считается по min/max ts и чувствителен к пропущеннымм событиям
